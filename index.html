Projects

SmartBrain

This project takes a photograph URL, and calls the Clarifai Face Detection API to detect the probability of a face in the picture, and sends that data back, where it is manipulated with CSS to display a face bounding box. React was used to create the components and UX, backend is psql database, served by Heroku

Life In The Wild

This project takes a series of images provided by photosforclass.com, and positions them in a flexbox, highlighting mobile responsivenss, and beautiful design using only HTML and CSS design/animations

Picture In Picture

This project prompts a user to select a video stream from their browser, and assembles it using Javascript into a viewing box which can be moved/resized accordsing to user needs, and can be placed onto and over other content to continue watching the stream while no longer on the same window.
