The Picture-In-Picture project allows a user to select a video stream from any open window or browser tab on their desktop and play it in a second window which is moveable and responsive to sizing, and can be placed on top of other content for a 'picture in picture' effect. On load you will be prompted to select a stream, and when ready hit the start button to launch PIP. This project was created with simple JavaScript, and DOM manipulation.

Robo-Joke-Teller

Click for a live demo!

Robo-Joke-Teller is a simple web app made with HTML and Javascript. When the 'Tell me a joke' button is pressed, the JokeAPI is called, and a programming joke is returned. This joke is then read by a second API, VoiceRSS, which uses a text-to-speech program to read the new joke. All jokes are Safe For Work!
