Joke-Teller-Bot

Web app demonstrating API usage. When the 'Tell me a joke' button is pressed, the JokeAPI is called, which returns a programming joke. This data is then issued to the VoiceRSS API, which will take the supplied data and read out the joke. All jokes are prgramming related, and Safe For Work! (Be sure to check your device's volume settings though to avoid startles.)
