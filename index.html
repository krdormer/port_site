Picture-In-Picture

Click for a live demo!

The Picture-In-Picture project allows a user to select a video stream from any open window on their desktop and play it in a second window which is moveable and responsive to sizing, and can be placed on top of other content for a 'picture in picture' effect. Simply open the project by following the link, and you will be prompted to select where the desired stream is located. (If not auto-prompted on page load, please simply refresh your browser tab and the prompt should display.) Once selected, hit the 'Start' button, and the PIP window will popup showing the selected video stream. This project was created with simple JavaScript, and DOM manipulation.

Robo-Joke-Teller

Click for a live demo!

Robo-Joke-Teller is a simple web app made with HTML and Javascript. When the 'Tell me a joke' button is pressed, the JokeAPI is called, and a programming joke is returned. This joke is then read by a second API, VoiceRSS, which uses a text-to-speech program to read the new joke. All jokes are Safe For Work!
